# Contributing to Sri Maruthi Teeth Empire

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/sri-maruthi-teeth-empire.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes thoroughly
6. Commit your changes: `git commit -m "Add your meaningful commit message"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Create a Pull Request

## Code Style

- Follow the existing code style
- Use TypeScript for all new files
- Use Tailwind CSS for styling
- Follow React best practices
- Write meaningful commit messages

## Testing

- Test your changes on different screen sizes
- Ensure all pages load correctly
- Check for console errors
- Verify routing works properly

## Pull Request Guidelines

- Provide a clear description of the changes
- Reference any related issues
- Include screenshots for UI changes
- Ensure all tests pass
- Keep PRs focused on a single feature or fix

## Questions?

If you have any questions, feel free to open an issue or contact the maintainers.
